Notes for ex-1-3

1. Understanding of the Problem

This exercise focuses on packaging and publishing the "helloworld" library and its CLI to TestPyPI (dev-pypi). The goal is to make the CLI installable using pip install.

2. Approach Taken

  • Configured a pyproject.toml file with dependencies.
  • Used Poetry to package the project.
  • Published the package to TestPyPI using: 
    poetry publish --build -r testpypi
  • Installed the package from TestPyPI and tested it.
